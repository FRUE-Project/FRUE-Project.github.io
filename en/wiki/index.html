Welcome to the FRUE Wiki!

This is the FRUE Wiki, where you can find information about the FRUE Project and its projects.
Below are links to the FRUE Projects:

fpkg is a free and open source package manager for Linux and FreeBSD. Its Wiki page provides helpful information about fpkg.

openFRUE is a Linux distribution based on Gentoo Linux, with the fpkg package manager included. Its Wiki page provides helpful information about openFRUE.
