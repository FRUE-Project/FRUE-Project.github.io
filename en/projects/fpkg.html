fpkg

FPKG is a highly versatile package manager for installing and managing FRUEpackages (.fpkg)

FPKG is a package manager for managing packages distributed in the FRUEpackage format (.fpkg). It is designed to be highly versatile and easy to use. It is currently in development and is not recommended for use in production environments. For more detailed information, see the wiki page. For the source code, see the GitHub repository.

FPKG is only supported on Linux and FreeBSD. It has not yet been tested on macOS or other BSD variants. To use FPKG, you must have the following dependencies installed: bash, wget, and maybe curl.
